Step 1: Include the Widget Script

Add the Dr. Nephro widget JavaScript file to your website. You can either:

Option A: Link to the file directly

<script src="path/to/nephro_widget.js"></script>

Option B: Copy the script content inline

<script> // Copy the entire content of nephro_widget.js here </script>

Step 2: Initialize the Widget

Add this JavaScript code to initialize the widget:

<script> // Initialize Dr. Nephro Widget document.addEventListener('DOMContentLoaded', function() { new NephroWidget({ apiUrl: 'http://localhost:8002', // Your API server URL position: 'bottom-right', // 'bottom-right' or 'bottom-left' theme: 'default' // Widget theme }); }); </script>

Step 3: Complete HTML Example

Here's a complete example of how to integrate the widget:

<!DOCTYPE html> <html> <head> <title>My Website with Dr. Nephro</title> </head> <body> <!-- Your website content --> <h1>Welcome to My Healthcare Website</h1> <p>Your regular website content goes here...</p> <!-- Dr. Nephro Widget Integration --> <script src="nephro_widget.js"></script> <script> document.addEventListener('DOMContentLoaded', function() { new NephroWidget({ apiUrl: 'http://localhost:8002', position: 'bottom-right', theme: 'default' }); }); </script> </body> </html>

Step 4: Configuration Options

Customize the widget behavior with these options:

new NephroWidget({ apiUrl: 'http://localhost:8002', // Required: Your API server URL position: 'bottom-right', // Optional: 'bottom-right' or 'bottom-left' theme: 'default', // Optional: Widget theme containerId: 'custom-container' // Optional: Custom container ID });
